# Indoor Air Quality Monitoring and Control System
**Smart Building Technology | Academic Project | UTCN**

## Project Overview
This project focuses on creating an intelligent IoT system designed to monitor indoor air quality parameters in real-time. It uses automated feedback loops to control ventilation systems, ensuring a healthy environment while optimizing energy consumption in Smart Buildings.

## Key Features
* **Real-time Monitoring:** Tracking CO2 levels, humidity, and temperature using hardware sensors.
* **Automated Control:** Implemented logic for ventilation activation based on threshold-driven feedback.
* **Knowledge-Based Systems:** Explored intelligent algorithms to optimize building automation.
* **IoT Integration:** Connected hardware components with software logic for a complete end-to-end solution.

## Technical Stack
* **Languages:** C++, C#
* **Hardware:** Sensors (CO2, Humidity, Temperature), Actuators (Ventilation Fans)
* **Concepts:** Knowledge-Based Systems, IoT Architecture, Feedback Control
